Project 2: Gradebook Program

Description

A C-based program that manages student grades using custom data structures. It allows users to add, remove, update, search, and display student records, calculates weighted averages, identifies missing grades, and exports data to a file. The system supports up to 5 students, uses a menu-driven interface, and stores student IDs, names, and three grade categories (homework, quiz, exam).

Skills Learned

  1. Data Structure Design and Organization
    • Create hierarchical structs with typedef
    • Design nested data structures where gradebook_t contains an array of student_t structs
    • Group related data fields (ID, name, homework, quiz, exam averages) into logical units
    • Understand memory layout of structs and how they've been passed between functions
    • Implement fixed-sized array storage for manageable data sets
  2. Memory Management and State Tracking
    • Using sentinel values (-1.0) to represent missing/uninitialized data
    • Implementing a counter variable to track active array elements
    • Shifting array elements during deletion to maintian contiguous storage
    • Preventing array overflow with boundary checking
    • Passing pointers to functions to modify original data rather than copies
  3. Algorithm Design and Implementation
    • Finding students by ID across the array (linear search algorithm)
    • Shifting elements during removal operations (array manipulation)
    • Creating non-repeating random numbers with validation
    • Determining missing grades and weighted averages
    • Checking for valid student IDs before operations
  4. Program Architecture and Flow Control
    • Main menu loop that conintues until exit condition
    • Nested function calls within menu options
    • Multiple exit poitns from functions based on conditions
    • State tracking across multiple functions using shared data structures
    • Error handling with appropriate user feedback
  5. CRUD Operations Implementation
    • Create: Adding new students with generated IDs
    • Read: Displaying students, searching by ID, listing all entries
    • Update: Modifying grade information for existing students
    • Delete: Removing students and shifting array elements
    • Complete lifecycle management of data records
